XMindConsolidate

Simple workload consolidation in an Xmind map

To install plugin

  • export it as jar
  • copy it to \Commons\plugins
  • update \XMind_Windows\configuration\config.ini
  • update \XMind_Windows\configuration\org.eclipse.equinox.simpleconfigurator\bundles.info

To avoid cache problems

  • delete \XMind_Windows\configuration\org.eclipse.osgi
  • delete \data

Start xmind
