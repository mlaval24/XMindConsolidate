XMindConsolidate

Simple workload consolidation in an Xmind map

To install plugin

  • find plugin directory of your Xmind installation ( on windows : C:\Program Files (x86)\XMind\plugins )
  • download jar file from github XMindConsolidate_0.0.6.201609301947.jar and copy to it
  • restart xmind

To avoid cache problems

  • delete %appdata%\Roaming\XMind\configuration-cathy_win32\org.eclipse.osgi
  • restart xmind
